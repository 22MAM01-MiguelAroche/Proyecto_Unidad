Etiquetas HTML

Estructura básica

html: Elemento raíz de cualquier documento HTML.
<html>...</html>

head: Contiene metadatos del documento, como título y enlaces a CSS.
<head>...</head>

title: Define el título de la página, que aparece en la pestaña del navegador.
<title>Título de la página</title>

body: Contiene el contenido visible de la página web.
<body>...</body>

Elementos de texto

h1 - h6: Encabezados, del más importante (h1) al menos importante (h6).
<h1>Encabezado</h1> ... <h6>Encabezado menor</h6>

p: Define un párrafo de texto.
<p>Este es un párrafo.</p>

strong: Texto con énfasis fuerte (normalmente en negrita).
<strong>Texto importante</strong>

em: Texto enfatizado (normalmente en cursiva).
<em>Texto enfatizado</em>

br: Inserta un salto de línea (no tiene etiqueta de cierre).
Texto<br>Nueva línea

Enlaces e imágenes

a: Define un enlace a otra página o recurso.
<a href="url">Texto del enlace</a>

img: Muestra una imagen. Usa el atributo src (no tiene etiqueta de cierre).
<img src="imagen.jpg" alt="Descripción">

Listas

ul: Lista desordenada.
<ul>...</ul>

ol: Lista ordenada.
<ol>...</ol>

li: Elemento de lista. Solo se usa dentro de ul o ol.
<li>Elemento de lista</li>

Contenedores

div: Contenedor genérico para agrupar contenido.
<div>Contenido agrupado</div>

span: Contenedor en línea, útil para aplicar estilos.
<span style="color:red;">Texto</span>

Formularios

form: Agrupa elementos de formulario para enviar datos.
<form action="procesar.php" method="post">...</form>

input: Campo de entrada de datos. Puede ser texto, checkbox, etc. (no tiene etiqueta de cierre).
<input type="text" name="nombre">

label: Etiqueta descriptiva para un campo del formulario.
<label for="nombre">Nombre:</label>

button: Botón que puede ejecutar acciones o enviar formularios.
<button type="submit">Enviar</button>

Recursos externos

link: Enlace a recursos externos (no tiene etiqueta de cierre).
<link rel="stylesheet" href="estilos.css">
